Procedure for Processing Reimbursements Request

Please have the following items when logging in.

  • Your Username
  • Password
  • Receipt information
  • Event Details

The complete list of qualify and non-qualify expense reimbursement allowance, is located on the employee and manager page respectively.

If your submission is complete and accurate, please allow up to 5 business days for reimbursement to be processed.

Once approved, an email will be sent.

Employee Specific Procedures

As a reminder, all expenses reimbursements must be Qualify Expenses which include the following.

  • Travel Business Purposes for employee only
  • Conference Fees, mileage per Google maps, and parking
  • Per Diem per the U.S. General Services Admin for accommodations and meals.

For further instructions, please see appropriate section for employee.

For any special or unique situations, please obtain prior written approval.

Managers Specific Procedures

As a reminder to all managers, all expenses reimbursements submitted must have project name included.

Please be sure and obtain all the following information prior to submission.

  • Usernameand password
  • Project Description with the purposes.
  • Lastly,Client(s) Name(s) of Project.

For any special or unique situations, please include the change order request invoice number.

For further instructions, please see appropriate section for managers.
